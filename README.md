# MiniCouncil
매월 23만원을 낼 수 없는 자들의 Model Council.

원래는 19개의 고오급 모델이 Opus의 지휘 하에 동작한다고 알려져 있으나,
우리는 그럴 여력이 없습니다.

그래서 만들었음.

---

## 필요한 것
1. 도커 기반으로 돌아가는데 알아서 잘 하시리라고 생각함
2. OpenRouter의 API 키가 필요합니다.

---

## .env 파일 예시
```dotenv
DISCORD_TOKEN= 디스코드 봇 토큰
OPENROUTER_KEY= 오픈라우터 API 키

MY_GUILD= 테스트용 서버 (선택 사항)
OWNER_ID= 오너 아이디 (선택사항)
```
밑에 두 놈은 쓰는 코드가 있긴 한데 동작하진 않음.

## 참여 모델
의장 겸 코미: 
- `google/gemini-3.1-pro-preview`

그리고 의문의 사료 친구들:

- `openai/gpt-5-mini` 친절하고 상세한 가이드 전문가
- `anthropic/claude-haiku-4.5` 논리적 허점을 찾아내는 비판적 분석가
- `perplexity/sonar` 최신 정보를 샅샅이 뒤지는 실시간 검색 전문가
- `x-ai/grok-4.1-fast` 실시간 여론과 트렌드를 읽는 유행 민감 전문가
